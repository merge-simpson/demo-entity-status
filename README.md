## Installation

### Git Clone

**Prerequisites**

- git (if cloning)
- docker

**Run this project.**

- Clone or download this project.  
    ```bash
    git clone ...
    ```
- Install and run database container using Docker Compose.
  ```bash
  # run this commandline at the root of this project.
  docker compose up -d
  ```
